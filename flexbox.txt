
1. Make a flex container

    -flex-wrap: nowrap / wrap / wrap-reverse
    X-justify-content: flex-start / flex-end / center / space-between / space-around / space-evenly
    Y-align-items: stretch / flex-start / flex-end / center / baseline
    -align-content: stretch / flex-start / flex-end / center / space-between / space-around
2. Flex elements are called = "Flex Items"
    -align-self: auto / stretch / flex-start / flex-end / center / baseline
    -order: 0 | <int>
    -flex-grow: 0 | <int>
    -flex-shrink: 1 | <int>
    -flex-basis: auto | <int>
3. Two dimensions
    - X: "Main Axis"
    - Y: "Cross Axis"
